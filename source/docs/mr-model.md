# &lt;mr-model&gt;

`<mr-model>` is used to load 3D model files.

## Example

```html
<mr-app>
  <mr-model src="..."></mr-model>
</mr-app>
```

## Definition and Usage

`<mr-model>` has one required attribute:

* `src`: the path to the target model file

## Supported file types

* GLB/GLTF
* STL
* OBJ
* USDZ
* ?
